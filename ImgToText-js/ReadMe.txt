# OCR (Optical Character Recognition)  
A web-based OCR application that extracts text from images using Tesseract.js and allows conversion of images to PDF.  

## Features  
- Upload an image and extract text using Tesseract.js  
- Convert images to PDF  
- Supports various image formats (JPG, PNG, etc.)  
- Simple and user-friendly interface  

## Tech Stack  
- HTML  
- CSS  
- JavaScript  
- [Tesseract.js](https://github.com/naptha/tesseract.js) (OCR Engine)
