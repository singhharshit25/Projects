# Tic-Tac-Toe  

A simple Tic-Tac-Toe game where two players take turns marking a 3x3 grid to achieve three in a row.  

## Features  
- Classic 3x3 Tic-Tac-Toe gameplay  
- Two-player mode  
- Simple UI for easy interaction  
- Detects winner and handles draw scenarios  

## How to Play  
1. Player 1 (X) and Player 2 (O) take turns marking a cell.  
2. The first player to align three symbols in a row, column, or diagonal wins.  
3. If all cells are filled and no one wins, it's a draw.
